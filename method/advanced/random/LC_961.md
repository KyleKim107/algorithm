> Question

* In a array nums of size 2 * n, there are n + 1 unique elements, and exactly one of these elements is repeated n times
* Return the element repeated n times

```txt
Input: nums[2,1,2,5,3,2]
Output: 2
```

{% endtab %}
{% tab title='LC_961.py' %}

```py
def repeatedNTimes(self, A):
  while 1:
    s = random.sample(A, 2)
    if s[0] == s[1]:
      return s[0]
```
