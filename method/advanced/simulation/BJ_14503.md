> Question

* Count number of cells cleaned by robot

```txt
Input:
3 3
1 1 0
1 1 1
1 0 1
1 1 1
Output:
1
```

{% endtab %}
{% tab title='BJ_14503.py' %}

```py
N, M, r, c, d = map(int,(input()+' '+input()).split())
G = [list(map(int,input().split()))for _ in range(N)]
x, y = [0,1,0,-1], [-1,0,1,0]
while 1:
  G[r][c]=2;t=0
  for _ in range(4):
    d = (d - 1) % 4
    if G[r+y[d]][c+x[d]]==0:
      t = 1
      r += y[d]
      c += x[d]
      break
  if t:
    pass
  elif G[r+y[d-2]][c+x[d-2]] == 1:
    break
  else:
    r += y[d-2]
    c += x[d-2]
print(sum(l.count(2)for l in G))
```
