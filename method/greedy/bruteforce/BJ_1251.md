* 세 단어로 나누기 : ar / rest / ed
* 각각 뒤집기 : ra / tser / de
* 합치기 : ratserde
* 단어가 주어지면, 이렇게 만들 수 있는 단어 중에서 사전순으로 가장 앞서는 단어를 출력하는 프로그램을 작성하시오

Input: mobitel
Output: bometil

```py
import sys
s = sys.stdin.readline().rstrip()
arr = []
for i in range(0,len(s)-2):
  for j in range(i+1,len(s)-1):
    a = s[0:i+1][::-1]
    b = s[i+1:j+1][::-1]
    c = s[j+1:][::-1]
    arr.append(a+b+c)
arr.sort()
print(arr[0])
```
