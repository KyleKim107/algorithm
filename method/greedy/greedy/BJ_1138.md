> Question

* Heights of N people are given
* Given people remember how many people were taller on the left, prints out how to line them up

```txt
Input:
4
2 1 1 0

Output: 4 2 1 3
```

{% endtab %}
{% tab title='BJ_1138.py' %}

```py
n = int(input())
a = list(map(int, input().split()) )
b = []
for i in range(n):
  b.insert(a[n - 1 - i], n - i)
print(*b)
```
