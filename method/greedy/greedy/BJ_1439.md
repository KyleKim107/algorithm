> Question

* You can grab one or more consecutive numbers and flip them all
* How many times do we have to make them have all the numbers?

```txt
Input: 0001100
Output: 1
```

{% endtab %}
{% tab title='BJ_1439.py' %}

```py
S = input()
count = 0
for i in range(len(S)-1):
  if S[i] != S[i+1]:
    count += 1
print((count + 1) // 2)
```
