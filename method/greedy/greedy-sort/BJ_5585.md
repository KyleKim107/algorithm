> Question

* Find minimum number of coins (500, 100, 50, 10, 5, 1) to pay n

```txt
Input: 380
Output: 4
```

{% endtab %}
{% tab title='BJ_5585.py' %}

```py
ret = 0
for coin in [500, 100, 50, 10, 5, 1]:
  while (coin <= n):
    n -= coin
    ret+=1
print(ret)
```
