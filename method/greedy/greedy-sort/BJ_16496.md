> Question

* Print maximum number with concatnation in any order

```txt
Input:
5
3 30 34 5 9

Output: 9534330
```

{% endtab %}
{% tab title='BJ_16496.py' %}

```py
n = input()
nums = sorted(input().split(), key=lambda x:x*9, reverse=True)
print(0 if nums[0] == "0" else ''.join(nums))
```
