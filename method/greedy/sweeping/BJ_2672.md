> Question

* Print total area covered by following rectangles

```txt
Input:
4
52.7 22.9 27.3 13.3
68.8 57.6 23.2 8.0
20.0 48.0 37.0 23.5
41.5 36.2 27.3 21.4

Output: 1853.61
```

{% endtab %}
{% tab title='BJ_2672.py' %}

```py
x = [[] for _ in range(10 ** 4)]
for _ in range(int(input())):
  l = [int(float(x) * 10) for n in input().split())]
  for i in range(l[0], l[0] + l[2]):
    x[i] += [(l[1], l[1] + l[3])]

s = 0
for i in x:
  if not i: continue
  i.sort()
  r=0
  for j in i:
    if j[1] <= r: continue
    s += j[1] - max(r,j[0])
    r = j[1]
print(s // 100 if s % 100 == 0 else s / 100)
```
