{% tab title='BJ_2160.py' %}

```py
N = int(input())
G = [[input() for _ in range(5)] for _ in range(N)]
m=36
for i in range(N-1):
  for j in range(i+1,N):
    d=0
    for k in range(5):
      for l in range(7):
        if G[i][k][l]!=G[j][k][l]:
          d+=1
    if m>d:
      m=d
      i1,j1=i,j
print(i1+1,j1+1)
```
