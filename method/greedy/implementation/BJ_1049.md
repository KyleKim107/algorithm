> Question

* Print minimum cost to buy to fix N string
* There are M brands, each with cost for 6 string and 1 string

```txt
Input:
10 3
20 8
40 7
60 4

Output: 36
```

{% endtab %}
{% tab title='BJ_1049.py' %}

```py
n, m = map(int, input().split())
c=[]
d=[]
for i in range(m):
  a, b = map(int, input().split())
  c.append(a)
  d.append(b)
e, f = min(c), min(d)
print(min([(n // 6 * e) + n % 6 * f, n // 6 * e + e, n * f]))
```
