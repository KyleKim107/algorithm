* Find length of Nth triangle

```txt
Input:
2
6
12

Output:
3
6
```

{% endtab %}
{% tab title='BJ_9461.py' %}

```py
N = int(input())
dp = [0, 1, 1]
for i in range(2, 102):
  dp.append(dp[-1] + dp[-3])

for _ in range(N):
  print(dp[int(input())])
```
