> Question

* Find sum of all xor substring

```txt
Input:
2
1 2

Output: 6   # 1 + 2 + 3
```

{% endtab %}
{% tab title='BJ_13710.py' %}

```py
total = 0
input()
arr = list(map(int, input().split()))
for i in range(30):
  c_odd = 0
  odd = 0
  for j in range(len(arr)):
    if arr[j] & (1 << i) > 0:
      odd = ~odd
    if odd:
      c_odd += 1
  for j in range(len(arr)):
    total += (2 ** i) * c_odd
    if (arr[j] & (1 << i)) > 0:
      c_odd = len(arr) - j - c_odd
print(total)
```
