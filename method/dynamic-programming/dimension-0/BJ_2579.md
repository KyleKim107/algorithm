> Question

* You cannot step three consecutive coins
* Print maximum coin you can get when you must eat the last coin

```txt
Input:
6
10
20
15
25
10
20

Output: 75
```

{% endtab %}
{% tab title='BJ_2579.py' %}

```py
a = b = c = 0
for i in range(int(input())):
  n = input()
  a, b, c = max(b, c), a + n, b + n
print(max(b,c))
```
