> Question

* When given the stock price for n days, print the longest rising day

```txt
Input:
6
5 2 1 4 5 3
3
1 1 1
4
4 3 2 1

Output:
3
1
1
```

{% endtab %}
{% tab title='BJ_3745.py' %}

```py
import sys
import bisect
li = sys.stdin.readlines()
for i in range(0, len(li), 2):
  D = []
  for e in map(int, li[i + 1].split()):
    if len(D) == 0 or e > D[-1]:
      D += [e]
    else:
      D[bisect.bisect_left(D, e)] = e
  print(len(D))
```
