> Question

* String is rotating with repetition
* Print shortest possible strings

```txt
Input:
5
aaaaa

Output: 1
```

{% endtab %}
{% tab title='BJ_1305.py' %}

```py
N = int(input())
S = input()

pi = [0] * N
j = 0
for i in range(1, N):
  while j > 0 and S[i] != S[j]:
    j = pi[j - 1]
  if S[i] == S[j]:
    j += 1
    pi[i] = j

print(N - pi[-1])
```
