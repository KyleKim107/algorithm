> Question

* Outputs the longest length of string that appears at least twice in the input

```txt
Input: abcdabcabb
Output: 3  # abc abc
```

{% endtab %}
{% tab title='BJ_1701.py' %}

```py
P = input()
def getPi(st):
  j = 0
  pi = [0] * len(st)
  for i in range(1, len(pi)):
    while j > 0 and st[i] != st[j]:
      j = pi[j-1]
    if st[i] == st[j]:
      j += 1
      pi[i] = j
  return max(pi)
res = 0
for i in range(len(P)):
  res = max(getPi(P[i:]), res)
print(res)
```
