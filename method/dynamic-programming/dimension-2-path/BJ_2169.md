> Question

* Robot can move left, right, down, find maximum sum

```txt
Input:
5 5
10 25 7 8 13
68 24 -78 63 32
12 -69 100 -29 -25
-16 -22 -57 -33 99
7 -76 -11 77 15

Output:
319
```

{% endtab %}
{% tab title='BJ_2169.py' %}

```py
N, M = map(int, input().split())
G = [list(map(int, input().split())) for _ in range(N)]
S, dp = 0, []
for i in range(M):
  S += G[0][i]
  dp.append(S)

for j in range(1, N):
  L = [dp[0] + G[j][0]] + [0] * (M - 1)
  for i in range(1, M):
    L[i] = G[j][i] + max(L[i - 1], dp[i])
  R = [0] * (M - 1) + dp[-1] + G[j][-1]
  for i in range(1, M):
    R[-i-1] = G[j][-i-1] + max(R[-i], dp[-i-1])
  for i in range(M):
    dp[i] = max(L[i], R[i])
print(dp[-1])
```
