* Find first bad version using isBadVersion function

50
isBadVersion False
25
isBadVersion True

```py
def firstBadVersion(self, n):
  lo, hi = 1, n
  while lo < hi:
    mi = (lo + hi) // 2
    if not isBadVersion(mi):
      lo = mi + 1
    else:
      hi = mi
  return lo
```
