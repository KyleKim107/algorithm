> Question

* N points above the two-dimensional plane are given
* Arrange coordinates in the order that (y-coordinate, x-coordinate)

```txt
Input:
5
0 4
1 2
1 -1
2 2
3 3

Output:
1 -1
1 2
2 2
3 3
0 4
```

{% endtab %}
{% tab title='BJ_11651.py' %}

```py
n = int(input())
li = []
for _ in range(n):
  a, b = map(int, input().split())
  li.append((b, a))
for a, b in sorted(li):
  print(b, a)
```
