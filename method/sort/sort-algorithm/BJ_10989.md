> Question

* Given N numbers print in ascending order

```txt
Input:
10
5
2
3
1
4
2
3
5
1
7

Output:
1
1
2
2
3
3
4
5
5
7
```

{% endtab %}
{% tab title='BJ_10989.py' %}

```py
import sys
a = int(input())
dic = {}
for _ in range(a):
  n = int(sys.stdin.readline())
  if n in dic:
    dic[n] += 1
  else:
    dic[n] = 1
for n in sorted(dic.keys()):
  for _ in range(dic[n]):
    print(n)
```
