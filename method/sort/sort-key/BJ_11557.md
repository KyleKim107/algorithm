> Question

* print university with most alchohol consumption

```txt
Input:
2
3
Yonsei 10
Korea 10000000
Ewha 20
2
Yonsei 1
Korea 10000000

Output:
Korea
Korea
```

{% tab title='BJ_11557.py' %}

```py
T = int(input())
for _ in range(T):
  N = int(input())
  alcohol = []
  for _ in range(N):
    S, L = map(str, input().split())
    alcohol.append([S,int(L)])
  alcohol = sorted(alcohol, key = lambda x: x[1])
  print(alcohol[-1][0])
```
