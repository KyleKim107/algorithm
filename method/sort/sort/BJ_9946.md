> Question

* For each line, check if two string are anagram

```txt
Input:
testing
intestg
abc
aabbbcccc
abcabcbcc
aabbbcccc
abc
xyz
END
END

Output:
Case 1: same
Case 2: different
Case 3: same
Case 4: different
```

{% endtab %}
{% tab title='BJ_9946.py' %}

```py
i = 1
while True:
  a, b = input(), input()
  if a == b == 'END':
    break
  if sorted(a) == sorted(b):
    print(f'Case {i}: same')
  else:
    print(f'Case {i}: different')
  i += 1
```
