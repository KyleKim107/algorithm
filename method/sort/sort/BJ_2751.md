> Question

* Print list in ascending order

```txt
Input:
5
5
2
3
4
1

Output:
1
2
3
4
5
```

{% endtab %}
{% tab title='BJ_2751.py' %}

```py
import sys
input = sys.stdin.readline
li = [int(input()) for _ in range(int(input()))]
for n in sorted(li):
  print(n)
```
