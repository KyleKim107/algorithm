> Question

* Find a number in the list that is exactly twice as many as you are

```txt
Input:
1 4 3 2 9 7 18 22 0
2 4 8 10 0
7 5 11 13 1 3 0
-1

Output:
3
2
0
```

{% endtab %}
{% tab title='BJ_4641.py' %}

```py
while True:
  a = list(map(int, input().split()))
  if a[0] == -1:
    break
  a_len = len(a)
  cnt = 0
  for i in range(a_len - 1):
    if a[i] * 2 in a:
      cnt += 1
  print(cnt)
```
