> Question

* Given an English sentence and an alphabet, counts how many times the alphabet appears in the sentence

```txt
Input:
g Programming Contest
n New Zealand
x This is quite a simple problem.
#

Output:
g 2
n 2
x 0
```

{% endtab %}
{% tab title='BJ_2386.py' %}

```py
while 1:
  x = input().lower()
  if x=='#':
    break
  print(x[0], x.count(x[0])-1)
```
