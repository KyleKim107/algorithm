> Question

* Print a + b until both are 0

```txt
Input:
1 1
2 3
3 4
9 8
5 2
0 0

Output:
2
5
7
17
7
```

{% endtab %}
{% tab title='BJ_10952.sh' %}

```sh
while read a b; do
  echo $((a+b))
done
```

{% endtab %}
{% tab title='BJ_10952.py' %}

```py
while True:
  a, b = map(int, input().split())
  if a == b == 0:
    break
  else:
    print(a + b)
```
