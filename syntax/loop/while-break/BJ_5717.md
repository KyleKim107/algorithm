> Question

* Print a + b until a and b are both 0

```txt
Input:
2 2
2 3
5 5
1 1
0 0

Output:
4
5
10
2
```

{% endtab %}
{% tab title='BJ_5717.py' %}

```py
while True:
  a, b = map(int, input().split())
  if a == b == 0:
    break
  else:
    print(a + b)
```
