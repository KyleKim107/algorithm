* Print denominator and numberator of Nth number

```txt
Input: 5
Output: 2 2
```

{% endtab %}
{% tab title='BJ_14723.py' %}

```py
n=int(input())
i=1
while n>i:
  n -= i
  i += 1
print(i+1-n,n)
```
