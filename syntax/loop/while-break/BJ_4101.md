> Question

* If the first number is greater than the second number, output Yes, or No one by one in a row

```txt
Input:
3 2
1 5
0 0

Output:
Yes
No
```

{% endtab %}
{% tab title='BJ_4101.py' %}

```py
while True:
  a, b = map(int, input().split())
  if a == b == 0:
    break
  if a > b:
    print("Yes")
  else:
    print("No")
```
