* Remove nth character from string

```txt
Input:
4
4 MISSPELL
1 PROGRAMMING
7 CONTEST
3 BALLOON

Output:
MISPELL
ROGRAMMING
CONTES
BALOON
```

{% tab title='BJ_2711.py' %}

```py
for _ in range(int(input())):
  a, b = input().split()
  print(b[:int(a) - 1]+b[int(a):])
```
