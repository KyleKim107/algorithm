> Question

* Given sequence 122333.. and A, B, find the sum of the digits A to B in the sequence

```txt
Input: 3 7
Output: 15  # 1 2 (2 3 3 3 4)
```

{% endtab %}
{% tab title='BJ_1292.sh' %}

```sh
arr=(0)
for i in {1..45}; do
  for (( j=0; j<i; ++j )); do
    arr+=($i)
  done
done

read a b
sum=0
for (( i=a; i<=b; ++i )); do
  sum=$((sum+arr[i]))
done

echo $sum
```

number_list = []
