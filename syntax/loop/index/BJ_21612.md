> Question

* Given N, P is 5 * N - 400
* print P and 1 if P < 100, 0 if P == 100, -1 if P > 100

```txt
Input: 102

Output:
95
1
```

{% endtab %}
{% tab title='BJ_21612.py' %}

```py
P = 5 * int(input()) - 400
print(P)
print([-1,0,1][(P<=100)+(P<100)])
```
