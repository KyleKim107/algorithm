> Question

* Keep round from one th digit, and print results of these operations

```txt
Input: 2045
Output: 2100  # 2050 -> 2100
```

{% endtab %}
{% tab title='BJ_2033.py' %}

```py
N = int(input())
d = 10
while N > d:
  if N % d >= d // 2:
    N += d
  N -= (N % d)
  d *= 10
print(N)
```
