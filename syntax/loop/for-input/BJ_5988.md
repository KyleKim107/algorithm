> Question

* For each line print oddity

```txt
Input:
2
1
4

Output:
odd
even
```

{% endtab %}
{% tab title='BJ_5988.py' %}

```py
N = int(input())
for _ in range(N):
  if int(input()) % 2 == 0:
    print("even")
  else:
    print("odd")
```
