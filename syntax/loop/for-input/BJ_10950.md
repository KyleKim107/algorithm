> Question

* Solve a + b for N lines

```txt
Input:
2
1 2
-1 2

Output:
3
1
```

{% endtab %}
{% tab title='BJ_10950.sh' %}

```sh
read t
for((;t--;));do
  read a b;echo $((a+b))
done
```

{% endtab %}
{% tab title='BJ_10950.py' %}

```py
for _ in range(int(input())):
  a, b = map(int, input().split())
  print(a + b)
```
