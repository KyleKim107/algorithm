> Question

* N Score is added when N consecutive number is right

```txt
Input:
5
OOXXOXXOOO
OOXXOOXXOO
OXOXOXOXOXOXOX
OOOOOOOOOO
OOOOXOOOOXOOOOX

Output:
10
9
7
55
30
```

{% endtab %}
{% tab title='BJ_8958.py' %}

```py
N = int(input())
for _ in range(N):
  st = input()
  score, row = 0, 0
  for ch in st:
    if ch == 'X':
      row = 0
    else:
      row += 1
      score += row
  print(score)
```
