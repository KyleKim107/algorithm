> Question

* Given distance, and n spaceship with fuel efficiency and max speed, find number of spaceships that can travel distance

```txt
Input:
2
3 100
52 75 10
88 13 44
56 9 5
2 920368
950 950 1
943 976 1

Output:
2
1
```

{% endtab %}
{% tab title='BJ_9501.py' %}

```py
t = int(input())
for i in range(t):
  n, d = map(int, input().split())
  result = 0
  for j in range(n):
    v, f, c = map(int, input().split())
    if f / c * v >= d: result += 1
  print(result)
```
