> Question

* For each test case, find sum of quotient divided by right number

> Question

```txt
Input:
2
3 2
4 5 7
3 8
4 5 7
Output:
7
0
```

{% endtab %}
{% tab title='BJ_9550.py' %}

```py
for _ in range(int(input())):
  N, K = map(int,input().split())
  answer = 0
  for i in list(map(int, input().split())):
    answer += i // K
  print(answer)
```
