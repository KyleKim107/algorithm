> Question

* Given Rock Paper Scissors, print winner

```txt
Input:
3
2
R P
S R
3
P P
R S
S R
1
P R

Output:
Player 2
TIE
Player 1
```

{% endtab %}
{% tab title='BJ_4493.py' %}

```py
for t in range(int(input())):
  n = int(input())
  a = 0
  for i in range(n):
    b = input()
    if b in ('P R','R S','S P'):
      a += 1
    elif b in ('R P','S R','P S'):
      a -= 1
  if a < 0:
    print("Player 2")
  elif a > 0:
    print("Player 1")
  else:
    print("TIE")
```
