> Question

* Find number of jobs can be finished in with given time

```txt
Input:
6 180
45 30 55 20 80 20

Output: 4
```

{% endtab %}
{% tab title='BJ_10409.py' %}

```py
n, T = map(int, input().split())
times = list(map(int, input().split()))
count = 0
for time in times:
  T -= time
  if T < 0:
    break
  count += 1
print(count)
```
