> Question

* Given 3, 5kg bags, find fewest number of bags

```txt
Input: 18
Output: 4
```

{% endtab %}
{% tab title='BJ_2839.py' %}

```py
N = int(input())
for i in range(0, N + 1, 3):
  if (N - i) % 5 == 0:
    print(i // 3 + (N - i) // 5)
    break
else:
  print(-1)
```
