> Question

* Print sum of lists for n test

```txt
Input:
2
1 5 3
1 7

Output:
9
8
```

{% endtab %}
{% tab title='BJ_11024.py' %}

```py
N = int(input())
for _ in range(N):
  n = list(map(int, input().split()))
  print(sum(n))
```
