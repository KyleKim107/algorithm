> Question

* Write a program to find the sum of the natural numbers included in S

```txt
Input:
10,20,
3
0,50
,1
00

Output: 210
```

{% endtab %}
{% tab title='BJ_10823.py' %}

```py
import sys
print(sum(eval(sys.stdin.read().replace("\n",""))))
```
