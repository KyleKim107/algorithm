> Question

* If the given formula is correct, output "correction" and "wrong answer" if incorrect

```txt
Input:
4
3 * 2 = 6
11 + 11 = 11
7 - 9 = -2
3 * 0 = 0

Output:
correct
wrong answer
correct
correct
```

{% endtab %}
{% tab title='BJ_13420.py' %}

```py
for _ in range(int(input())):
  eq, ans = input().split('=')
  print("correct" if eval(eq) == int(ans) else "wrong answer")
```
