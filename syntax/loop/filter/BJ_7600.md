> Question

* Print how many different alphabets are written in each line

```txt
Input:
The quick brown fox jumped over the lazy dogs.
2 + 2 = 4
New Zealand Programming Contest.
#

Output:
26
0
16
```

{% endtab %}
{% tab title='BJ_7600.py' %}

```py
while a := input().lower() != '#':
  print(len(set(filter(lambda x: x.isalpha(), a))))
```
