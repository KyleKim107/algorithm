> Question

* Print I LOVE CBNU if even

```txt
Input:
3

Output:
***
 *
* *
```

{% endtab %}
{% tab title='BJ_14624.py' %}

```py
n=int(input())
if n%2:
  print("*"*n)
  print(" "*(n//2)+"*")
  for i in range(1,n//2+1):print(" "*(n//2-i)+"*"+" "*(i*2-1)+"*")
else:print("I LOVE CBNU")
```
