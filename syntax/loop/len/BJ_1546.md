> Question

* 자기 점수 중에 최댓값을 골랐다. 이 값을 M이라고 한다. 그리고 나서 모든 점수를 점수/M*100으로 고쳤다
* 성적을 위의 방법대로 새로 계산했을 때, 새로운 평균을 구하는 프로그램을 작성하시오

```txt
Input:
3
40 80 60

Output: 75.0
```

{% endtab %}
{% tab title='BJ_1546.py' %}

```py
n = int(input())
li = list(map(int, input().split()))
print(sum(li) / max(li) * 100 / len(li))
```
