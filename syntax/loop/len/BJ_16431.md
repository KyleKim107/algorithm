* Print who can go to destination first

```txt
Input:
3 5
1 1
2 3

Output: bessie
```

{% endtab %}
{% tab title='BJ_16431.py' %}

```py
a, b = map(int, input().split())
c, d = map(int, input().split())
e, f = map(int, input().split())
x = max(abs(e - a), abs(f - b))
y = abs(e - c) + abs(f - d)
if x == y:
  print('tie')
elif x < y:
  print('bessie')
else:
  print('daisy')
```
