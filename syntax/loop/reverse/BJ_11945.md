> Question

* For each line print reversed string

```txt
Input:
5 7
0010000
0111010
1111111
0111010
0010000

Output:
0000100
0101110
1111111
0101110
0000100
```

```py
n, m = map(int, input().split())
for i in range(n):
  print(input()[::-1])
```
