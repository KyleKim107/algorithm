> Question

* For each line, a given number outputs a palindromic sleep "yes" or "no"

```txt
Input:
121
1231
12421
0

Output:
yes
no
yes
```

{% endtab %}
{% tab title='BJ_1259.cpp' %}

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  string s;
  while (cin>>s) {
    if(s=="0") break;
    string t = s;
    reverse(t.begin(), t.end());
    cout << (s == t ? "yes" : "no")<<endl;
  }
  return 0;
}
```

```py
while True:
  n = input()
  if n == '0':
    break
  if n == n[::-1]:
    print("yes")
  else:
    print("no")
```
