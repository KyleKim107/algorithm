> Question

* Print length of longest increasing subarray

```txt
Input:
8
12 20 1 3 4 4 11 1

Output: 8
```

{% endtab %}
{% tab title='BJ_2846.py' %}

```py
input()
m, n, l = 0, 999999999999, 999999999999
for i in map(int, input().split()):
  if l < i:
    m = max(m, i-n)
  else:
    n = i
  l = i
print(m)
```
