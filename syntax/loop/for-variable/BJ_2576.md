> Question

* Find odd's sum and minimum

```txt
Input:
12
77
38
41
53
92
85

Output:
256
41
```

{% endtab %}
{% tab title='BJ_2576.py' %}

```py
sm = 0
mn = 100
for _ in range(7):
  a = int(input())
  if a % 2 == 1:
    mn = min(mn, a)
    sm += a
if mn == 100:
  print(-1)
else:
  print(sm)
  print(mn)
```
