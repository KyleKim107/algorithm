> Question

* Print total sum of subtraction to make the list decreasing

```txt
Input:
4
5
3
7
5

Output: 6
```

{% endtab %}
{% tab title='BJ_2847.py' %}

```py
score = [int(input()) for i in range(int(input()))][::-1]
cur = 20000
ans = 0
for s in score:
  ans += max(0, s - cur + 1)
  cur = min(cur - 1, s)
print(ans)
```
