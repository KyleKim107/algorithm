> Question

* print # times each number from 0 to 9 was written in the result of calculating A × B × C

150

427

3
1
0
2

{% tab title='BJ_2577.cpp' %}

```cpp
#include<cstdio>
int arr[10];
int main() {
  int s=1, t;

  for(int i=0;i<3;++i) {
    scanf("%d",&t);
    s*=t;
  }
  while(s) {
    ++arr[s%10];
    s/=10;
  }

  for(int i=0;i<10;++i)
    printf("%d\n",arr[i]);
}
```
