> Question

* Print 2 if 2 appears more than e else e
* if 2 appears same as e print yee

```txt
Input:
12
222eee222eee

Output: yee
```

{% endtab %}
{% tab title='BJ_17094.py' %}

```py
input()
s = input()
if s.count('e') > s.count('2'):
  print("e")
elif s.count('e') < s.count('2'):
  print("2")
else:
  print('yee')
```
