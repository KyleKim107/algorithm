{% tab title='BJ_4673.py' %}

```py
li = [True] * 10090
def d(n):
  ret = n
  while n:
    ret += n % 10
    n //= 10
  return ret

for n in range(10000):
  li[d(n)] = False

for i in range(10001):
  if li[i]:
    print(i)
n = int(input())
```
