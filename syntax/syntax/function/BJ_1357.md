> Question

* find rev(rev(a) + rev(b))

```txt
Input: 123 100
Output: 223
```

{% endtab %}
{% tab titlepyJ_1357.md'' %}

```py
a, b = map(int, input().split())
def rev(a):
  return int("".join(reversed(str(a))))
print(rev(rev(a) + rev(b)))
```
