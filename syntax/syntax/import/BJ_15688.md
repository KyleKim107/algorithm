> Question

* When given N numbers, sort them in descending order

```txt
Input:
5
5
4
3
2
1

Output:
1
2
3
4
5
```

{% endtab %}
{% tab title='BJ_15688.py' %}

```py
import sys
l = []
for _ in range(int(sys.stdin.readline())):
  l.append(int(sys.stdin.readline()))
print("\n".join(map(str, sorted(l))))
```
