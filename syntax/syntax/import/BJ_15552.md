> Question

* Print a + b for each line

```txt
Input:
5
1 1
12 34
5 500
40 60
1000 1000

Output:
2
46
505
100
2000
```

{% tab title='BJ_15552.cpp' %}

```cpp
#include <iostream>
using namespace std;
int main() {
  cin.tie(NULL);
  ios::sync_with_stdio(false);
  int T;
  cin >> T;
  while (T--) {
    int A, B;
    cin >> A >> B;
    cout << A + B << '\n';
  }
}
```

{% tab title='BJ_15552.py' %}

```py
import sys
input = sys.stdin.readline
n_test = int(input())
for _ in range(n_test):
  a, b = map(int, input().split())
  print(a + b)
```
