> Question

* Print list in reversed order

```txt
Input:
5
1
2
3
4
5

Output:
5
4
3
2
1
```

{% endtab %}
{% tab title='BJ_11931.py' %}

```py
import sys
input = sys.stdin.readline
n = int(input())
li = [int(input()) for _ in range(n)]
for e in sorted(li, reverse=True):
  print(e)
```
