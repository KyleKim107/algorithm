> Question

* How many squares with side c needed to cover rectangle with side a, b

```txt
Input: 7 9 3
Output: 9
```

{% endtab %}
{% tab title='BJ_13136.py' %}

```py
import math
a, b, c = map(int, input().split())
print(math.ceil(a / c) * math.ceil(b / c))
```
