> Question

* Given courses and score print GPA

```txt
Input:
2
4
3 4.3
2 2.0
4 0.0
2 4.0
3
4 0.0
4 0.0
3 0.0

Output:
11 2.3
11 0.0
```

{% tab title='BJ_10984.py' %}

```py
for _ in range(int(input())):
  a = int(input())
  h, p = 0, 0
  for i in range(a):
    c, g = map(str, input().split())
    h += int(c)
    p += float(c) * float(g)
  p = round(p / h, 1)
  print(h, p)
```
