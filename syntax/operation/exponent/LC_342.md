* Print if power if 4

```txt
Output: True
```

{% tab title='LC_342.cpp' %}

```cpp
// (4^n - 1) % 3 == 0
// 4^n - 1 = (2^n + 1) * (2^n - 1) one  must be a multiple of 3
bool isPowerOfFour(int num) {
  return num > 0 && (num & (num - 1)) == 0 && (num - 1) % 3 == 0;
}
```
