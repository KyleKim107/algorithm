> Question

* If N files can be compressed and each file can have a maximum bit length of b, print "yes" and "no" if not

```txt
Input: 13 3
Output: yes  # 2 ** (3 + 1) = 14
```

{% endtab %}
{% tab title='BJ_5043.py' %}

```py
a, b = map(int, input().split())
print("yes" if a < 2 ** (b + 1) else "no")
```
