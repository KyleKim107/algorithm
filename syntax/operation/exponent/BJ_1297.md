> Question

* Given, diagonal width height ratio, find height and width
* If the height or width of the actual TV is a decimal point, floor it

```txt
52 9 16
```

{% endtab %}
{% tab title='BJ_1297.py' %}

```py
d, w, h = map(int, input().split())
x = (d ** 2 / (w ** 2 + h ** 2)) ** 0.5
print(int(x * w), int(x * h))
```
