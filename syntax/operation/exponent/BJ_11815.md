> Question

* print 1 if square else 0

```txt
Input:
2
4 3

Output: 1 0
```

{% endtab %}
{% tab title='BJ_11815.py' %}

```py
input()

for n in map(int, input().split()):
  if n == (int(n ** 0.5) ** 2):
    print(1, end = " ")
  else:
    print(0, end = " ")
```
