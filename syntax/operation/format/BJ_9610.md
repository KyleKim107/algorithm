> Question

* Count number of points on each coordinate and axis

```txt
Input:
5
0 0
0 1
1 1
3 -3
2 2

Output:
Q1: 2
Q2: 0
Q3: 0
Q4: 1
AXIS: 2
```

{% endtab %}
{% tab title='BJ_9610.py' %}

```py
q1 = q2 = q3 = q4 = axis = 0
N = int(input())
for _ in range(N):
  x, y = map(int, input().split())
  if x == 0 or y == 0:
    axis += 1
  elif x > 0 and y > 0:
    q1 += 1
  elif x < 0 and y > 0:
    q2 += 1
  elif x < 0 and y < 0:
    q3 += 1
  elif x > 0 and y < 0:
    q4 += 1
print(f"Q1: {q1}", f"Q2: {q2}", f"Q3: {q3}" f"Q4: {q4}" f"AXIS: {axis}", sep="\n")
```
