> Question

* Print Hello World, Judge i, n times

```txt
Input: 3

Output:
Hello World, Judge 1!
Hello World, Judge 2!
Hello World, Judge 3!
```

{% endtab %}
{% tab title='BJ_9316.py' %}

```py
n = int(input())
for i in range(1, n + 1):
  print(f"Hello World, Judge {i}!")
```
