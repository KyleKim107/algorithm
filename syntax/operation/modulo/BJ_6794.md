> Question

* 첫 줄에 n 이 주어진다 다음 n * 2 줄에 a, b 가 주어 질때
* a 나누기 b의 몫과 나머지를 각각 출력하라

```txt
Input:
2
4 2
5 2

Output:
2
0

2
1

```

{% endtab %}
{% tab title='BJ_9498.py' %}

```py
n = int(input())
for i in range(n):
  a, b = int(input()), int(input())
  print(a // b)
  print(a % b)
  print()
```
