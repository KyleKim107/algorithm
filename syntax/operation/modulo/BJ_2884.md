> Question

* print time 45m ago

```txt
Input: 10 10
Output: 9 25
```

{% endtab %}
{% tab title='BJ_2884.sh' %}

```sh
read a b
t=$((60*a+b-45))
if [ $t -lt 0 ]; then
  t=$((t+1440))
fi
echo $((t/60)) $((t%60))
```

{% endtab %}
{% tab title='BJ_2884.py' %}

```py
h, m = map(int, input().split())
m = h * 60 + m - 45
if m < 0:
  m += 1440
print(m // 60, m % 60)
```
