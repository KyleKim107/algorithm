> Question

* with 5, 1 minute, 10 second button, find the minimum button operation method to meet the given cooking time of T seconds

```txt
Input: 100
Output: 0 1 4
```

{% endtab %}
{% tab title='BJ_10162.py' %}

```py
n = int(input())
if n % 10 != 0:
  print(-1)
else:
  print(n // 300, n % 300 // 60, n % 60 // 10)
```
