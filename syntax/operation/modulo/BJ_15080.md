> Question

* Print time difference

```txt
Input:
10 : 0 : 0
11 : 0 : 0

Output: 3600
```

{% endtab %}
{% tab title='BJ_15080.py' %}

```py
a, b, c = map(int, input().split(" : "))
d, e, f = map(int, input().split(" : "))
print(((d - a) * 3600 + (e - b) * 60 + f - c + 24 * 3600) % (24 * 3600))
```
