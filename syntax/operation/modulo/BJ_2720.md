> Question

* Given integer, print number of exchanges in Quarter, Dime, Nickel, Penny

```txt
Input:
3
124
25
194

Output:
4 2 0 4
1 0 0 0
7 1 1 4
```

{% endtab %}
{% tab title='BJ_2720.py' %}

```py
for i in range(int(input())):
  x = int(input())
  print(x // 25, x % 25 // 10, x % 25 % 10 // 5, x % 25 % 10 % 5)
```
