> Question

* a, b, c 가 주어질 때, a + 1 와 b + 1 의 곱을 c + 1로 나눈 몫에 1를 뺀 수를 출력하라

```txt
Input: 15 18 11
Output: 24
```

{% endtab %}
{% tab title='BJ_9498.py' %}

```py
a, b, c = map(int, input().split())
print((a + 1) * (b + 1) // (c + 1) - 1)
```
