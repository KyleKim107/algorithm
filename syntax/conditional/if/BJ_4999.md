> Question

* print "no" if second line is longer else "go"

```txt
Input:
aaah
ah

Output: go
```

{% endtab %}
{% tab title='BJ_4999.py' %}

```py
a = input()
b = input()
print("no" if len(b) > len(a) else "go")
```

{% endtab %}
{% tab title='BJ_4999.sh' %}

```sh
read a
read b
if [ ${#b} -gt ${#a} ]; then
  echo 'no'
else
  echo 'go'
fi
```
