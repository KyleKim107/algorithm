> Question

* width and height is a, height is 4
* When cutting cake at width b, height c, print largest pieces

```txt
Input: 10 4 7
Output: 168
```

{% endtab %}
{% tab title='BJ_17874.py' %}

```py
a, b, c = map(int, input().split())
print(max(a - b, b) * max(a - c, c) * 4)
```
