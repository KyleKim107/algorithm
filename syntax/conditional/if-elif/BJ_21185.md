> Question

* Pick n numbers between 1 to 100
* print oddity of the sum of n numbers, print Either if both are possible

```txt
Input: 2
Output: Odd
```

{% endtab %}
{% tab title='BJ_21185.py' %}

```py
n = int(input())
if n % 4 == 0:
  print('Even')
elif n % 2 == 0:
  print('Odd')
else:
  print('Either')
```
