> Question

* If all three numbers are same print (sum of all) * 10000 + 1000
* else if only two of them are same print (sum of two) * 100 + 1000
* else print max * 100

```txt
Input: 3 3 6
Output: 1300
```

{% endtab %}
{% tab title='BJ_2480.py' %}

```py
a, b, c = map(int, input().split())
if a == b == c:
  print(10000 + a * 1000)
elif a == b or a == c:
  print(1000 + a * 100)
elif b == c:
  print(1000 + c * 100)
else:
  print(max(a, b, c) * 100)
```
