> Question

* print age, Korean age, year age

```txt
Input:
2003 3 5
2003 4 5

Output:
0
1
0
```

{% endtab %}
{% tab title='BJ_16199.py' %}

```py
y1, m1, d1 = map(int, input().split())
y2, m2, d2 = map(int, input().split())

k_age = y2 - y1 + 1
if (m1, d1) > (m2, d2):
  age = k_age - 2
else:
  age = k_age - 1
print(age)
print(k_age)
print(k_age - 1)
```
