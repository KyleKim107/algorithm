> Question

* print White if (a is smaller or equal to 50) and (b is smaller or equal to 10)
* print Red if q is greater than 30
* print Yellow otherwise

```txt
Input:
50
10

Output: White
```

{% endtab %}
{% tab title='BJ_20673.py' %}

```py
p = int(input())
q = int(input())
if p <= 50 and q <= 10:
  print("White")
elif q > 30:
  print("Red")
else:
  print("Yellow")
```
