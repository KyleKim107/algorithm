> Question

* Given two rectangles, check if a rect fit to other with margin of 1

```txt
Input: 30 30 27 27
Output: 1
```

{% endtab %}
{% tab title='BJ_21591.py' %}

```py
w1, h1, w2, h2 = map(int, input().split())
print(int(w2 + 2 <= w1 and h2 + 2 <= h1))
```
