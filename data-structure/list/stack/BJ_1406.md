> Question

* Simultate typing with cursor
  * L, R : move cursor left, right
  * B : Remove char from left
  * P $ : add $ to left

```txt
Input:
abcd
3
P x
L
P y

Output: abcdyx
```

{% endtab %}
{% tab title='BJ_1406.py' %}

```py
import sys
l, r = list(input()), []
input()
for c in sys.stdin:
  if c[0]=='L':
    if l: r.append(l.pop())
  elif c[0]=='D':
    if r: l.append(r.pop())
  elif c[0]=='B':
    if l: l.pop()
  else:
    l.append(c[2])
print(''.join(l + r[::-1]))
```
