> Question

* Print if input has valid parenthesis

```txt
Input:
6
(())())
(((()())()
(()())((()))
((()()(()))(((())))()
()()()()(()()())()
(()((())()(

Output:
NO
NO
YES
NO
YES
NO
```

{% endtab %}
{% tab title='BJ_9012.py' %}

```py
for T in range(int(input())):
  f, a, s = 1, 0, input()
  for i in s:
    if i=="(":
      a+=1
    else:
      a-=1
    if a<0:
      f=0
  print("YES" if f and a==0 else "NO")
```
