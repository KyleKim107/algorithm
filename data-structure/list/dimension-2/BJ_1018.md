> Question

* minimum number of squares that need to be repainted to make as checker board

```txt
Input:
10 13
BBBBBBBBWBWBW
BBBBBBBBBWBWB
BBBBBBBBWBWBW
BBBBBBBBBWBWB
BBBBBBBBWBWBW
BBBBBBBBBWBWB
BBBBBBBBWBWBW
BBBBBBBBBWBWB
WWWWWWWWWWBWB
WWWWWWWWWWBWB

Output: 12
```

```py
def block_cnt(r, c):
  cntWB, cntBW = 0, 0
  for i in range(r, r+8):
    for j in range(c, c+8):
      if (i - r + j - c) % 2 == 0:
        if G[i][j] == 'B':  cntWB += 1
        else:               cntBW += 1
      else:
        if G[i][j] == 'W':  cntWB += 1
        else:               cntBW += 1
  return min(cntWB, cntBW)
min_change = N * M
for i in range(N-7):
  for j in range(M-7):
    min_change = min(min_change, block_cnt(i, j))

print(min_change)
```
