> Question

* When words are given on the blackboard, print the letters in the order they are read vertically

```txt
Input:
ABCDE
abcde
01234
FGHIJ
fghij

Output: Aa0FfBb1GgCc2HhDd3IiEe4Jj
```

{% endtab %}
{% tab title='BJ_10798.py' %}

```py
li = [input() for _ in range(5)]
for i in range(15):
  for j in range(5):
    if i < len(li[j]):
      print(li[j][i], end='')
```
