> Question

* Find sum of given 2d array

```txt
Input:
2
1 2 3
-1 -3 2

Output: 0
```

{% endtab %}
{% tab title='BJ_16504.cpp' %}

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int N; cin >> N;
  long long ans = 0;
  for (int i = 0; i < N; i++) {
    for (int j = 0; j < N; j++) {
      int x; cin >> x;
      ans += x;
    }
  }
  cout << ans << "\n";
}
```
