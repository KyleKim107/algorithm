> Question

* Given list of match results (a won b)
* Determine the number of match needed to find exact rank for each player

```txt
Input:
6
5
1 2
2 3
3 4
5 4
6 5

Output:
2
2
2
0
3
3
```

{% endtab %}
{% tab title='BJ_10026.py' %}

```py
INF = 1E9

n = int(input())
m = int(input())
G = [[INF] * n for _ in range(n)]
for i in range(n):
  G[i][i] = 0

for _ in range(m):
  a, b = map(int, input().split())
  G[b - 1][a - 1] = 1

for k in range(n):
  for i in range(n):
    for j in range(n):
      G[i][j] = min(G[i][j], G[i][k] + G[k][j])

result = []
for i in range(n):
  cnt = 0
  for j in range(n):
    if G[j][i] == INF and G[i][j] == INF:
      cnt += 1
  result.append(cnt)

print(*result, sep="\n")
```
