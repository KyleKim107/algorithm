> Quesiton

* Given pairs of compaison, sort them

```txt
Input:
3 2
1 3
2 3

Output: 1 2 3
```

{% endtab %}
{% tab title='' %}

```py
N, M = map(int, input().split())
G = [[] for i in range(N + 1)]
wait = [-1] + [0] * (N)
for _ in range(M):
  u, v = map(int, input().split())
  G[u].append(v)
  wait[v] += 1

bfs = [n for n in range(N + 1) if wait[n] == 0]
print(*bfs, end=' ')
for i in bfs:
  for j in G[i]:
    wait[j] -= 1
    if wait[j] == 0:
      print(j, end=' ')
      bfs.append(j)
```
