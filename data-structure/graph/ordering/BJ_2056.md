> Question

* print minimum number of time to finish all jobs

```txt
Input:
7
5 0
1 1 1
3 1 2
6 1 1
1 2 2 4
8 2 2 4
4 3 3 5 6

Output: 23
```

{% endtab %}
{% tab title='BJ_2056.py' %}

```py
cost = [0]*10001

for i in range(1, int(input())+1):
  cost[i], *prior = map(int, input().split())
  cost[i] += max(cost[p] for p in prior)

print(max(cost))
```
