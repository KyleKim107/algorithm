> Question

* Check if each line match to following patterns (100+1+|01)+

```txt
Input:
3
10010111
011000100110001
0110001011001

Output:
NO
NO
YES
```

{% endtab %}
{% tab title='BJ_1013.py' %}

```py
import re
for i in range(int(input())):
  print('YES'if re.fullmatch(r"(100+1+|01)+", input()) else 'NO')
```
