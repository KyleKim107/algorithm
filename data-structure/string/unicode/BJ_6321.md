> Question

* Print next unicode for each character

```txt
Input:
2
HAL
SWERC

Output:
String #1
IBM
String #2
TXFSD
```

{% tab title='BJ_6321.py' %}

```py
for c in range(1,int(input())+1):
  r = ''
  for i in input(): r += chr(65+(ord(i)-64)%26)
  print(f"String #{c}\n{r}\n")
```
