> Question

* Determine if a number can be divided into two parts, and product of front digit and product of back digit are same

```txt
Input: 1236
Output: YES
```

{% endtab %}
{% tab title='BJ_1356.py' %}

```py
n = input()
n_len = len(n)
true = 0
for i in range(n_len - 1):
  left = 1
  right = 1
  for j in range(i + 1):
    left *= int(n[j])
  for k in range(i + 1, n_len):
    right *= int(n[k])
  if left == right:
    print("YES")
    break
else:
  print("NO")
```
