> Question

* erase all CAMBRIDGE from letter

```txt
Input: LOVA
Output: LOV
```

{% endtab %}
{% tab title='BJ_2789.py' %}

```py
a = "CAMBRIDGE"
s = input()
for i in a:
  for j in range(len(s)):
    if i == s[j]:
      s[j] = ''
print(*s, sep='')
```
