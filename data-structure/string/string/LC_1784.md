> Question

* Given a binary string s ​​​​​without leading zeros, return if s contains at most one contiguous segment of ones

```txt
Input: "010"
Output: False
```

{% endtab %}
{% tab title='LC_1784.py' %}

```py
def checkOnesSegment(self, s: str) -> bool:
  return "01" not in s
```
