> Question

* Given letter score, print GPA

```txt
Input: A0
Output: 4.0
```

{% endtab %}
{% tab title='BJ_2754.cpp' %}

```cpp
#include <iostream>
using namespace std;

int main() {
  char a, b; cin >> a >> b;
  cout.precision(1);
  if (a != 'F')
    cout << fixed << (4 - (a - 'A') + (b == '+' ? 0.3 : (b == '0' ? 0 : -0.3)));
  else
    cout << "0.0";
}
```
