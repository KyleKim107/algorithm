> Question

* The factorial calculates the value of digit i as ai√ói!
* When given a number written in the factorial, obtain the value read in decimal

```txt
Input:
719
1
15
110
102
0

Output:
53
1
7
8
8
```

{% endtab %}
{% tab title='BJ_5692.py' %}

```py
import sys
input = sys.stdin.readline
base = [1, 2, 6, 24, 120]
while True:
  n = int(input())
  if not n:
    break
  factorial, result = 0, 0
  while n:
    result += ((n % 10) * base[factorial])
    factorial += 1
    n //= 10
  print(result)
```
