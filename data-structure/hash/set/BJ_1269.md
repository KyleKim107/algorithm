> Question

* Print number of unique element that doesn't exist in other list

```txt
Input:
3 5
1 2 4
2 3 4 5 6

Output: 4
```

{% endtab %}
{% tab title='BJ_1269.py' %}

```py
input()
a = set(map(int, input().split()))
b = set(map(int, input().split()))
print(len(a - b) + len(b - a))
```
